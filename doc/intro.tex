\addcontentsline{toc}{section}{\en{Intro}\\\ru{Введение}}

\en{This library\ --- yet another attempt to write ultraportable framework for
the development of software for embedded systems, from simple 8-bit
microcontrollers to full-fledged computers with general-purpose OS.}

\ru{Эта библиотека -- еще одна попытка написания максимально портабельного
фреймворка для разработки ПО для встраиваемых систем, от самых простых 8-битных
микроконтроллеров до полноценных компьютеров с ОС общего назначения.}
